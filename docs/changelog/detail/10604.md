# v1.6.4 详细更新内容预览

_2026-03-06_


## 优化注解窗口的表现

优化注解窗口（描述提示词窗口）在屏幕边缘以及多屏幕下的显示效果，同时修复消失位置可能会和鼠标位置重叠而导致的闪烁问题。

该更新有 3 项提交

- 修复闪烁问题  
  在此之前，鼠标移开注解控件后，注解窗口会立即隐藏，但下一次鼠标再次移动到注解控件上时，注解窗口出现的位置可能与鼠标位置重叠，导致反复触发移入移出事件，从而让窗口不停闪烁。现在的移出事件会将注解窗口移出整个虚拟屏幕，而非隐藏。
- 优化边缘显示效果  
  注解窗口默认在鼠标右下方显示，但当屏幕右下角空间不够时会改用从左、上方显示，该方式跳跃变化太大，现在改用缩进显示。
- 优化多屏幕显示效果  
  使用多屏幕时，注解窗口的位置判断会出现问题，但现在不会了。

## 移除更新检查的阻塞锁

现在更新检查不再会让程序阻塞，错误和阻断信息不再有效，除了那些烦人的消息窗口依旧会出现。`--noupdatecheck` 参数依旧可用。
